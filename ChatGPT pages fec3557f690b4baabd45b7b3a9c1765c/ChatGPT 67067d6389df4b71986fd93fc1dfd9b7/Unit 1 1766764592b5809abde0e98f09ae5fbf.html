Q1. What is the 'Training Set' and 'Test Set' in a Machine Learning Model? How Much Data will You Allocate for Your Training and Test Sets?

  • Training Set:
    • The part of the dataset used to train a machine learning model.
    • It helps the model learn patterns, relationships, and decision boundaries.
    • The model adjusts its parameters using this data.
  • Test Set:
    • Used to evaluate the performance of the trained model.
    • It acts as unseen data to check how well the model generalizes.
    • The model does not see this data during training.
  • Data Allocation:
    • Typically, data is split into:
      • 70-80% for training
      • 20-30% for testing
    • If there is a validation set, the split may be:
      • 60-70% training
      • 10-20% validation
      • 20% testing

Q2. Relate the Following Feature Transformation Types and Explain in Detail

  1. x=log⁡(x)x = \log(x) (Log Transformation):
    • Reduces skewness in data and normalizes distribution.
    • Example: Converting exponential growth data into a linear trend.
  1. x=xx = \sqrt{x} (Square Root Transformation):
    • Reduces right skewness, helpful for data with large variations.
    • Example: Used in financial data where high values dominate.
  1. x=3x+2x = 3x + 2 (Linear Transformation):
    • Preserves relationships but scales and shifts values.
    • Example: Adjusting temperature values from Celsius to Fahrenheit.
  1. x=1/xx = 1/x (Reciprocal Transformation):
    • Handles inversely proportional relationships, reducing large variations.
    • Example: Used in speed and time calculations.

Q5. Differentiate Between Supervised and Unsupervised Learning

AspectSupervised LearningUnsupervised Learning
DefinitionUses labeled data, where input-output pairs are known.Uses unlabeled data, where patterns and structures are learned without predefined labels.
ObjectiveLearns a mapping function from input to output.Finds hidden patterns, structures, or relationships in data.
Training DataContains both input (features) and output (labels).Contains only input (features) without predefined labels.
Techniques UsedClassification, Regression.Clustering, Dimensionality Reduction.
Example AlgorithmsDecision Trees, SVM, Neural Networks, Linear Regression.K-Means Clustering, DBSCAN, PCA, Hierarchical Clustering.
ApplicationsSpam email detection, Loan approval prediction, Disease diagnosis.Customer segmentation, Anomaly detection, Market Basket Analysis.

Supervised learning is like learning with a teacher (labels guide the model), while unsupervised learning is like self-learning (the model identifies patterns on its own).

Q7. What is Machine Learning? Explain the 4 Characteristics and Real-Life Examples

  • Definition:
    • "Machine Learning is a field of study that gives computers the capability to learn without being explicitly programmed."
  • 4 Characteristics:
    1. Automated Data Visualization - Helps businesses analyze patterns (e.g., customer behavior).
    1. Automation - Reduces manual efforts (e.g., self-driving cars).
    1. Customer Engagement - Improves user experience (e.g., chatbots).

Q8. Explain the Following Feature Selection Techniques with Examples

  1. Missing Value Ratio:
    • Removes features with a high percentage of missing values.
    • Example: If a dataset has 90% missing values in a column, it's removed.
  1. Fisher’s Score:
    • Measures how well a feature separates different classes.
    • Example: Used in image classification to rank pixel importance.
  1. Information Gain:
    • Determines feature importance based on entropy reduction.
    • Example: Used in Decision Trees to choose the best split.
